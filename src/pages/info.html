What is JUST Extension?

JUST Extension is an all-in-one Chrome extension specifically designed to enhance the browsing experience for students at Jordan University of Science and Technology.

Features

JUST Extension offers several helpful tools:

  • Dark theme for easier browsing
  • Automatic course evaluation filler
  • Course registration helper
  • GPA calculator (semester & cumulative)
  • Quick access to important university links
  • Auto-close notifications

How to Use

Simply click on the features you'd like to use from the main menu. Each tool is designed to be intuitive and user-friendly. For the GPA calculator and Register Helper, you can access them via the buttons on the main page.
